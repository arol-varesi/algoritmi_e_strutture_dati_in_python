# Algorithms and Data Structures in Python

Repository di appoggio per le attività pratiche del corso:

**ALGORITHMS AND DATA STRUCTURES IN PYTHON (3 DAYS)**

## Obiettivo

Fornire esempi, esercitazioni e soluzioni in Python per i principali concetti di algoritmi e strutture dati, con un approccio pratico e applicazioni reali.

## Argomenti trattati

### 1. Analisi della complessità

- Notazione Big-O per complessità temporale e spaziale
- Come analizzare algoritmi e operazioni comuni

### 2. Strutture lineari

- Array
- Stack
- Queue
- Operazioni tipiche e relative complessità

### 3. Tipi di base in Python

- `str`
- `list`
- `tuple`

### 4. Tipi avanzati in Python

- `queue`
- `collections.deque`
- `bisect`
- `array`
- `collections.Counter`

### 5. NumPy

- `ndarray`
- utilizzi principali per calcolo numerico e vettorializzazione

### 6. Applicazioni pratiche

- Ricerca
- Espressioni matematiche
- Sliding window

### 7. Strutture non lineari: insiemi e dizionari

- `set`
- `dict`
- `Counter`
- Operazioni e complessità

### 8. Strutture non lineari: alberi

- Operazioni e complessità sugli alberi
- Rappresentazione con OOP (`class Tree`, `class Node`)
- Rappresentazione senza OOP (liste annidate, dizionari)
- Visitazioni: preorder, inorder, postorder
- Applicazione: codifica di Huffman

### 9. Strutture non lineari: grafi

- Operazioni e complessità sui grafi
- Rappresentazione con OOP (`class Graph`, `class Node`)
- Rappresentazione senza OOP (liste di adiacenza/dizionari)
- Implementazioni di BFS e DFS
- Applicazioni: percorso minimo in grafo non pesato, TSP
